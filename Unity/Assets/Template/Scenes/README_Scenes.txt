###Scenes Folder###

	Stores all the Scenes files here. 

##What are Scene files?##

	Format: .unity
	Description: This folder contains all of the scene to be made with Unity.

##Categorize them!##

#Categories:#

	*Alpha: Prototype, not a game, Testing features, or Playing around.
	*Beta: Testing, "Play"-able demo.
	*Copula: Completed, Stable scenes. The Start Menus should be here first thing.
