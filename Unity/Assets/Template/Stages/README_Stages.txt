###Stages Folder###

	Stores all the Stage-related files here. 

##What are Stages?##

	A folder contain whatever elements, prefabs needed for a certain part of a scene. Each stage represents a certain unique theme (in art or in sound).

Each Stages should contain the following folders:

	* Animations
	* Behaviour Scripts
	* Common Prefabs
	* Documents
	* Environment Sprites

	Categories: 
	By Theme (Forest, Under water, etc...) or By whatever
